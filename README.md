# 📝 To-Do List App (React)

## 📌 Présentation

Je suis en train de développer une application de type **To-Do List** avec **React**.
Ce projet me permet de m’entraîner avec les bases de React, la gestion des composants, des états (state) et des événements.

L’objectif principal est de pouvoir ajouter, supprimer et gérer des tâches facilement via une interface simple et intuitive.

---

## 🚀 Fonctionnalités

Dans cette application, je peux :

* Ajouter une nouvelle tâche
* Supprimer une tâche
* Marquer une tâche comme terminée (si implémenté)
* Voir la liste des tâches en temps réel
* Gérer l’état des tâches avec React (`useState`)

## ▶️ Installation et exécution

Pour lancer le projet en local :

1. Cloner le repository :

```bash
git clone <url-du-projet>
```

2. Accéder au dossier :

```bash
cd nom-du-projet
```

3. Installer les dépendances :

```bash
npm install
```

4. Lancer le projet :

```bash
npm run dev
```
